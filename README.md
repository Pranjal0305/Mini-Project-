# 🚀 SEO AI Agent

An AI-powered full-stack SEO analysis web application that helps users analyze websites and receive actionable SEO insights using AI.

Built with:

- ⚛️ React + Vite + Tailwind CSS
- ⚡ FastAPI (Python)
- 🤖 Groq AI (llama-3.3-70b-versatile)

---

# ✨ Features

## 🔍 SEO Analysis Modes

### 🔧 Full Audit
Technical SEO analysis including:
- Meta tags
- Page speed
- Mobile optimization
- Crawlability
- Indexing issues

### 🤖 GEO (Generative Engine Optimization)
AI-focused SEO insights:
- Structured data
- E-E-A-T signals
- AI search visibility
- LLM optimization

### ⚔️ Competitor Analysis
Competitive SEO strategy:
- Keyword gaps
- Content opportunities
- Backlink ideas
- SERP analysis

### 🗂️ Cluster Strategy
Content clustering recommendations:
- Topic clusters
- Pillar pages
- Internal linking
- Semantic SEO structure
